## Overview
------------------
Stochastic implemenation for network-based covid-19 simulation based on [the work of Dr. Alison Hill](https://alhill.shinyapps.io/COVID19seir/) .



## Installation
------------------
With:
```console
pip install -r requirements.txt
```
## Example Usage
-----------------
With
```console
python simulation.py
```
